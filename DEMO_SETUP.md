### B. Write Journal Entries (2 minutes)

1. Click **"Journal"** in navigation
2. Create 3-4 entries:
   - **Entry 1**: "Today I practiced mindfulness and felt more centered."
   - **Entry 2**: "Had a good conversation with a friend. Social connection helps."
   - **Entry 3**: "Completed my daily walk goal. Small wins matter."

### C. Create Goals (1 minute)

1. Click **"Goals"** in navigation
2. Create 3 goals:
   - **Goal 1**: "Take a 10-minute walk daily" (Daily)
   - **Goal 2**: "Write in journal 3 times this week" (Weekly)
   - **Goal 3**: "Complete 20 mood entries this month" (Monthly)
3. Update progress on at least one goal

### D. Create Community Posts (1 minute)

## üêõ Troubleshooting

### "Database not connected"
- Check MySQL is running
- Verify database user exists
- Check `.env` file configuration

### "Can't create account"
- Check server logs
- Verify MySQL connection
- Check database permissions

### "Features not loading"
- Check browser console (F12)
- Verify authentication token
- Check network tab for API errors
